# МДК 02.02 · Кроссворд по транспортным сетям связи

Интерактивный кроссворд на 15 терминов из лекции по транспортным сетям (SDH/PDH/WDM/MPLS). Все определения — прямые цитаты из лекции с указанием номера слайда.

**Открыть:** https://timurmingazov06.github.io/mdk-crossword/

## Фичи

- Адаптив под десктоп и мобильный (sticky-полоска с активным вопросом над виртуальной клавиатурой)
- Звук на ввод буквы / правильное слово / финиш (можно отключить)
- Шаринг — снимок текущего состояния как PNG в буфер обмена или файлом
- Печать — A4 для раздачи ученикам (с ответами или без)
- Сохранение прогресса в localStorage

## Стек

Один самодостаточный `index.html` — без зависимостей, без сборки. CSS Grid + чистый JS + Web Audio API + Canvas API.

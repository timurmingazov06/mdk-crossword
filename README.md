# МДК 02.02 · Кроссворд по транспортным сетям связи

Интерактивный кроссворд на 15 терминов из лекции по транспортным сетям (SDH/PDH/WDM/MPLS). Все определения — прямые цитаты из лекции.

**Открыть:** https://timurmingazov06.github.io/mdk-crossword/

## Фичи

- Десктоп + мобильная версия (sticky-полоска с активным вопросом над виртуальной клавиатурой)
- На мобильном — нативная клавиатура отключена, используется встроенная кириллическая раскладка
- Свайп влево/вправо по нижней полоске → переключение слов
- Лёгкая вибрация на ввод буквы (на устройствах с поддержкой)
- Звук на ввод/правильное слово/финиш (можно отключить — 🔊 в шапке)
- Шаринг — PNG-снимок в буфер обмена (или файлом)
- Печать — A4 для раздачи ученикам (с ответами или без)
- Сохранение прогресса в localStorage

## Стек

Один самодостаточный `index.html` — без зависимостей, без сборки. CSS Grid + чистый JS + Web Audio API + Canvas API.

## Локальная разработка

```bash
python3 -m http.server 8765 --directory .
# http://localhost:8765
```

Чтобы пересобрать сетку с другими словами — отредактировать `build_grid_v3.py` (список `WORDS_V3`), запустить `python3 build_grid_v3.py`, затем встроить новую `grid_final.json` в `index.html` (блок `const DATA = ...`).
